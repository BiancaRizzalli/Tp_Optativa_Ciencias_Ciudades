# Trabajo Práctico Optativa: Ciencias de Datos para Ciudades

Este repositorio contiene tres trabajos prácticos desarrollados para la materia Ciencias de Datos para Ciudades, cuyo objetivo general es analizar patrones de movilidad urbana, características personales y emociones a partir de distintos conjuntos de datos.


## Contenido del Repositorio
Este repositorio incluye la realizacion de **tres trabajos prácticos (TP1, TP2 y TP3)** relacionados con procesamiento, análisis y visualización de datos en **R** 

- Un archivo .r para cada trabajo (TP1.r, TP2.r, TP3.r)
- Datasets de entrada (datos_Tp1.xlsx, datos_Tp2.xlsx, datos_Tp3.xlsx )
- Datasets de salida:
    - TP1 -> resultados_viajes.xlsx
    - TP3 -> resultados_emociones.xlsx
- Imagenes .png que representan graficos resultado de los analisis
- Informe PDF con los procedimientos, desiciones tomadas, y resultados del analisis

---

## Requisitos Previos

### 1. Instalar R
Para ejecutar los trabajos prácticos necesitás tener instalado **R**.

- Descargar R desde: https://cran.r-project.org/
- Seguir el instalador según tu sistema operativo (Windows, macOS o Linux).

**Opcional:** instalar **RStudio**, un entorno de desarrollo para trabajar cómodamente con scripts R:  
https://posit.co/download/rstudio-desktop/

---

## 2.Instalación de Librerías Necesarias

Antes de ejecutar cualquiera de los TPs, instalá las librerías requeridas.  
Ejecutá el siguiente código en R o RStudio:

```r
install.packages(c(
  "jsonlite",
  "dplyr",
  "data.table",
  "ggplot2",
  "readxl",
  "readr",
  "writexl"
))

```
